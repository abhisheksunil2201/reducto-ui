# Reducto UI Component Library

Reducto UI is a component library. It can be used to add components to your web app and make your development process faster.
## Tech Stack
- HTML
- CSS

## Components
- Alert
- Avatar
- Badge
- Button
- Card
- Grid
- Image
- Input
- List
- Modal
- Navigation
- Rating
- Slider
- Text Utilities
- Toast
